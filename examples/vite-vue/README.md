## Setup

```bash
npm install
npm run dev
```

## Vite Configuration

```js
export default {
  build: {
    target: 'esnext',
  },
}
```

## Usage

```html
<script setup>
import { onMounted } from 'vue'
import gradientGL from 'gradient-gl'

onMounted(() => {
  gradientGL('a2.eba9')
})
</script>

<template>
  <div id="app" />
</template>
```
